netns-vrf
=========

a linux netns wrapper
---------------------

###install

* required: ruby, linux 3.0.2+

* clone the repo:

    git clone https://github.com/nopedial/netns-vrf

* build the gem: 

    gem build netns-vrf.gemspec

* install the gem: 

    gem install ./netns-vrf-0.0.3.gem

* run: 

    netns-vrf
